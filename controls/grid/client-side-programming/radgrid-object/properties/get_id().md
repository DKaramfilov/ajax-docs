# get_id()



## 

Property which returns the id of the grid client object.


|  **get_id()**  |
| ------ |
||

Example:

````JavaScript
function getRadGridID() {
    var grid = $find("<%=RadGrid1.ClientID %>");
    var gridId = grid.get_id();            
}
````
