# OnItemSelecting


This event is fired when a TreeListDataItem is about to be selected on the client. This event can be canceled.


| | |
| ------ | ------ |
| **Fired by** | RadTreeList |
| **Arguments** | get_item() - the TreeListDataItem object which is being selected |
|| get_cancel() - bool |
|| set_cancel(bool) |
| **Can be canceled** | Yes |
