# RadToolBarButton Id and Custom Attributes (Q1 2016 SP1)

## Introduced changes

Since Q1 2016 SP1, due to changes in the **RadToolBar**'s rendering, ID and custom attributes can no longer be set to the **RadToolBarButton**.
